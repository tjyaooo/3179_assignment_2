Dominance of Google
Android
Android is a Mobile Operating System, created by Google which was released on September 2008. It boasts an open-source OS that is free and is modified based on the Linux Kernel (GeeksforGeeks, 2021). Created by Andy Rubin, it is writtten in many languages like Java, C, C++ and XML. Currently, there are more than 2 billion smart devices that use Google Android OS as their operating system. Chrome and Search
Chrome, an Internet browser released by Google in 2008, gained massive popularity at that time as it offered support for cloud computing, allowing users to access programs operating in the Internet. eg: Google Calendar and Google Spreadsheet through the browser. Additionally, it is the first browser to isolate different tabs as an entirely separate process, offering performance gains, at the cost of additional memory (Britannica, 2021).

Google Search, is a search engine. It is the most visited website in the world (Alexa, 2021). It is estimated that each person spends 17 minutes at google.com. Using the PageRank algorithm developed by Larry Page, It is able to measure the importance of website pages and display search results appropriately. The algorithm works by counting the number of quality links to a website and determine a rough estimate on the importance of the site. It is assumed that important websites will receive more links from other websites (Google, 2021).
