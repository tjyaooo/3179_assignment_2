Dominance of Google
Android
Android is a Mobile Operating System, created by Google which was released on September 2008. It boasts an open-source OS that is free and is modified based on the Linux Kernel (GeeksforGeeks, 2021). Created by Andy Rubin, it is writtten in many languages like Java, C, C++ and XML. Currently, there are more than 2 billion smart devices that use Google Android OS as their operating system. Chrome and Search
Chrome, an Internet browser released by Google in 2008, gained massive popularity at that time as it offered support for clour computing, allowing users to access programs operating in the Internet. eg: Google Calendar and Google Spreadsheet through the browser. Additionally, it is the first browser to isolate different tabs as an entirely separate process, offering performance gains, at the cost of additional memory (Britannica, 2021)
